# Unified Integrated Tax System – Sri Lanka

## Introduction
This project proposes a Unified Integrated Digital Tax System for Sri Lanka to improve tax compliance, reduce tax evasion, and increase government revenue by integrating Inland Revenue, Customs, and Excise departments into a single digital platform.

## Problem
Sri Lanka has a low tax-to-GDP ratio, high tax evasion, and inefficient manual tax administration systems.

## Solution
This project proposes:
- Unified tax portal
- TIN-based system
- E-invoicing system
- AI-based tax evasion detection
- GIS-based tax analysis
- Online tax payment system

## System Components
- Taxpayer Registration System
- Tax Calculation Engine
- Payment Gateway Integration
- Fraud Detection Model
- Data Analytics Dashboard
- GIS Tax Mapping

## Technologies Used
- Python
- SQL
- Machine Learning
- React
- Node.js
- GIS (QGIS)
- Power BI / Tableau
